# KOL 管理系統 - 使用者手冊

歡迎使用 KOL 管理系統！本手冊將引導您快速上手系統的各項功能。

## 🚀 快速開始

### 啟動系統
```bash
# 安裝依賴 (首次使用)
npm install

# 啟動開發伺服器
npm run dev
```

開啟瀏覽器訪問: **http://localhost:5173/**

---

## 📋 功能導覽

系統包含三大核心功能區：
1. **儀表板** - 查看統計數據與總覽
2. **KOL 列表** - 管理所有 KOL 資料
3. **合作管理** - 管理合作專案與銷售追蹤

---

## 1️⃣ 儀表板

### 功能說明
儀表板提供系統的整體概況，包括：
- 核心指標（KOL 數量、總粉絲數、平均互動率、進行中合作）
- 財務指標（總收益、總預算、ROI）
- 表現最佳 KOL 排行榜
- KOL 分類分布圖
- 社群平台統計
- 合作專案狀態

### 使用步驟
1. 點擊導航列的「**儀表板**」按鈕
2. 查看各項統計數據
3. 點擊排行榜中的 KOL 可快速查看詳細資料

### 重要指標說明
- **ROI (投資回報率)** = (總收益 - 總預算) / 總預算 × 100%
- **平均互動率** = 所有 KOL 的互動率平均值
- **進行中合作** = 狀態為「已確認」或「進行中」的專案數

## 2️⃣ KOL 列表

### 功能說明
KOL 列表是系統的核心功能，可以：
- 查看所有 KOL 的卡片式列表
- 快速搜尋與篩選 KOL
- 新增、編輯、刪除 KOL
- 查看 KOL 詳細資料

### 2.1 瀏覽 KOL 列表

**步驟**:
1. 點擊導航列的「**KOL 列表**」按鈕
2. 系統會以卡片形式顯示所有 KOL

**卡片資訊包含**:
- KOL 姓名與暱稱
- 評分 (星級)
- 內容分類標籤
- 社群平台粉絲數
- 總粉絲數
- 平均互動率
- 自訂標籤

### 2.2 搜尋與篩選 KOL

**搜尋功能**:
- 在搜尋框輸入關鍵字
- 可搜尋：KOL 姓名、暱稱、標籤
- 即時顯示搜尋結果

**篩選功能**:
- **分類篩選**: 選擇特定內容分類（美妝、3C、美食等）
- **平台篩選**: 選擇特定社群平台（YouTube、Instagram 等）
- 可同時使用搜尋與篩選

**清除篩選**:
- 如果找不到結果，點擊「清除篩選條件」重置

### 2.3 新增 KOL

**步驟**:
1. 點擊右上角的「**新增 KOL**」按鈕
2. 填寫基本資料（必填項目標示 *）
   - 真實姓名 *
   - 暱稱/藝名 *
   - Email *
   - 電話 *
   - 地區
   - 評分 (1-5，預設 5)

3. 選擇內容分類
   - 從下拉選單選擇分類
   - 點擊「+」按鈕新增
   - 可新增多個分類
   - 點擊標籤上的垃圾桶圖示移除

4. 選擇語言
   - 從下拉選單選擇語言
   - 點擊「+」按鈕新增
   - 可新增多種語言

5. 新增標籤
   - 輸入自訂標籤（如：高互動、專業、值得信賴）
   - 按 Enter 或點擊「+」按鈕新增
   - 點擊標籤上的垃圾桶圖示移除

6. 新增社群平台
   - 點擊「**新增平台**」按鈕
   - 選擇平台類型（YouTube、Instagram、Facebook、TikTok、Twitter）
   - 填寫：
     - 帳號名稱（例如：@username）
     - 網址
     - 粉絲數
     - 互動率 (%)
     - 平均觀看數（YouTube 專用）
   - 可新增多個平台帳號
   - 點擊垃圾桶圖示刪除平台

7. 填寫備註（選填）

8. 點擊「**新增**」按鈕儲存

### 2.4 編輯 KOL

**步驟**:
1. 在 KOL 卡片上點擊「編輯」圖示（鉛筆）
2. 修改需要更新的資料
3. 點擊「**更新**」按鈕儲存
4. 點擊「**取消**」放棄修改

### 2.5 刪除 KOL

**步驟**:
1. 在 KOL 卡片上點擊「刪除」圖示（垃圾桶）
2. 確認刪除操作
3. 系統會同時刪除該 KOL 相關的合作專案與銷售追蹤資料

⚠️ **注意**: 刪除操作無法復原，請謹慎操作！

### 2.6 查看 KOL 詳細資料

**步驟**:
1. 點擊 KOL 卡片上的「**查看**」按鈕
2. 系統會顯示 KOL 的完整資料頁面

**詳細資料包含**:
- 基本資料與聯絡資訊
- 評分與標籤
- 統計摘要（總粉絲數、平均互動率、總銷售額、總佣金）
- 社群平台資料卡片
- 合作專案歷史
  - 進行中的合作
  - 已完成的合作
  - 每個專案的銷售成效

**操作**:
- 點擊「**編輯**」按鈕快速編輯 KOL 資料
- 點擊「**返回列表**」回到列表頁
- 點擊社群平台網址可直接開啟該平台

## 3️⃣ 合作管理

### 功能說明
合作管理功能讓您可以：
- 建立與追蹤合作專案
- 查看專案狀態與預算
- 追蹤銷售成效

### 3.1 瀏覽合作專案

**步驟**:
1. 點擊導航列的「**合作管理**」按鈕
2. 系統會顯示所有合作專案

**專案資訊包含**:
- 專案名稱與品牌
- KOL 姓名
- 專案狀態（彩色標籤）
- 專案期間
- 預算與實際費用
- 合作平台
- 銷售成效（如果有）
  - 點擊數
  - 轉換數
  - 收益
  - 佣金

### 3.2 專案狀態說明

| 狀態 | 顏色 | 說明 |
|-----|------|------|
| **待確認** | 灰色 | 專案提案階段 |
| **洽談中** | 黃色 | 正在與 KOL 協商條件 |
| **已確認** | 紫色 | 雙方已同意合作條件 |
| **進行中** | 藍色 | 專案執行中 |
| **已完成** | 綠色 | 專案已完成 |
| **已取消** | 紅色 | 專案取消 |

### 3.3 搜尋與篩選專案

**搜尋功能**:
- 在搜尋框輸入關鍵字
- 可搜尋：專案名稱、品牌名稱、KOL 姓名

**狀態篩選**:
- 使用下拉選單選擇特定狀態
- 選擇「全部狀態」顯示所有專案

### 3.4 新增合作專案

**步驟**:
1. 點擊「**新增專案**」按鈕
2. 填寫專案資料（必填項目標示 *）
   - 選擇 KOL * （從下拉選單選擇）
   - 專案名稱 *
   - 品牌名稱 *
   - 狀態 *（預設：待確認）
   - 開始日期 *
   - 結束日期 *
   - 預算 (NT$) *
   - 實際費用 (NT$)
   - 備註

3. 點擊「**新增**」儲存專案

### 3.5 編輯合作專案

**步驟**:
1. 點擊專案卡片右上角的「編輯」圖示
2. 修改需要更新的資料
3. 更新專案狀態（例如從「洽談中」改為「已確認」）
4. 點擊「**更新**」儲存

### 3.6 刪除合作專案

**步驟**:
1. 點擊專案卡片右上角的「刪除」圖示
2. 確認刪除操作
3. 系統會同時刪除相關的銷售追蹤資料

⚠️ **注意**: 刪除操作無法復原！

### 3.7 查看銷售成效

已完成的專案如果有設定銷售追蹤，會在專案卡片下方顯示：
- **點擊數**: 折扣碼或聯盟連結的點擊次數
- **轉換數**: 實際購買的人數
- **收益**: 總銷售金額
- **佣金**: 支付給 KOL 的佣金
- **折扣碼**: 專屬折扣碼（如果有）

## 💡 使用技巧

### 技巧 1: 善用標籤系統
為 KOL 建立有意義的標籤，例如：
- 「高互動」- 互動率高的 KOL
- 「專業」- 內容專業度高
- 「穩定」- 合作穩定可靠
- 「快速回覆」- 溝通效率高

### 技巧 2: 定期更新社群數據
建議每月更新一次社群平台的粉絲數與互動率，以確保數據準確性。

### 技巧 3: 使用評分系統
根據合作經驗為 KOL 評分（1-5 星），方便日後快速篩選優質 KOL。

### 技巧 4: 詳細記錄備註
在 KOL 資料或合作專案中詳細記錄：
- 合作經驗
- 溝通偏好
- 特殊要求
- 注意事項

### 技巧 5: 追蹤專案狀態
定期更新合作專案的狀態，確保工作流程清晰。

## ❓ 常見問題

### Q1: 如何匯入大量 KOL 資料？
**A**: 目前系統需要手動新增 KOL。未來版本會支援 Excel 匯入功能。

### Q2: 資料會自動儲存嗎？
**A**: 目前資料儲存在瀏覽器的記憶體中，重新整理頁面會回到初始資料。未來版本會整合後端資料庫。

### Q3: 如何自動更新社群數據？
**A**: 目前需要手動更新。未來版本會整合社群平台 API，實現自動更新。

### Q4: 可以匯出報表嗎？
**A**: 目前尚未支援匯出功能。未來版本會支援匯出 PDF 與 Excel 報表。

### Q5: 支援多人協作嗎？
**A**: 目前為單機版本。未來版本會支援多人協作與權限管理。

### Q6: 如何計算 ROI？
**A**: ROI = (總收益 - 總預算) / 總預算 × 100%
