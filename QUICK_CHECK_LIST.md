## 🎯 快速測試指令集合

```bash
# 前端測試
npm run dev                    # 啟動前端開發伺服器
# 然後在瀏覽器開啟 http://localhost:5173 (或其他 port)

# 後端測試
cd backend
npm install                    # 安裝依賴
npm run migrate                # 執行資料庫遷移
npm run dev                    # 啟動後端伺服器

# API 測試
curl http://localhost:3000/api/health                      # 健康檢查
curl -X POST http://localhost:3000/api/auth/login \        # 登入
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}'

# 使用變數存 token (Bash)
TOKEN=$(curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' | jq -r '.data.token')

curl http://localhost:3000/api/kols \                      # 獲取 KOL
  -H "Authorization: Bearer $TOKEN"
```
